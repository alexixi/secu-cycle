# Frontend Mobile

## Prérequis
- [Node.js](https://nodejs.org/) installé sur votre machine.
- L'application **Expo Go** installée sur votre smartphone (iOS ou Android).

## Installer les dépendances
```sh
make install
```

## Développement
### Lancer le projet en mode développement
```sh
make dev
```
Ouvrez l'application Expo Go sur votre smartphone et scannez le QR code affiché dans le terminal pour voir l'application en action.
