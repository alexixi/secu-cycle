# Backend

## Prérequis
- [Docker](https://www.docker.com/)
- [Python](https://www.python.org/)

## Build les images Docker
```sh
make build
```

## Lancer les conteneurs Docker

### En premier plan
```sh
make up
```

### En arrière plan
```sh
make background
```

## Arrêter les conteneurs Docker
```sh
make down
```
